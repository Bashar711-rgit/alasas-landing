# منصة الأسس — صفحة إعلانية تفاعلية

صفحة هبوط إعلانية احترافية (عربية RTL) لمنصة «الأسس» للخدمات التعليمية المتكاملة.

## المزايا

- واجهة رئيسية متحركة مع شعار المنصة ورمز QR للتواصل عبر واتساب
- شريط خدمات متحرك + شبكة الخدمات الثماني بتأثيرات تفاعلية
- قسم «لماذا نحن» وقسم تواصل مع زر واتساب مباشر (+966578568011)
- متجاوبة بالكامل مع الجوال والحاسوب

## التقنيات

React + TypeScript + Vite + Tailwind CSS + Framer Motion + Lucide Icons

## التشغيل

```bash
npm install
npm run dev      # بيئة التطوير
npm run build    # بناء الإنتاج إلى dist/
```
